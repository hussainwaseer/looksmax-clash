# ⚔️ Looksmax Clash

**The Ultimate AI-Powered Aesthetic Arena.**

Looksmax Clash is a high-performance, real-time competitive platform where users go head-to-head in aesthetic analysis. Powered by **MediaPipe**, **Google Gemini AI**, and **Socket.io**, it provides non-sugarcoated, geometric facial scoring and a global ranking system.

![Looksmax Clash Preview](public/icon-512.png)

## 🚀 Key Features

### 👁️ Real-Time Battle Arena
- **Face-Off Mode:** Real-time 1v1 battles with live video feeds via WebRTC.
- **Spectator Mode:** Watch the top moggers in real-time without joining the fight.
- **Matchmaking:** Global queue to find opponents instantly.
- **Mogging Meter:** Dynamic UI that shifts as the AI detects dominance.

### 📈 Pro-Level Analytics
- **Geometric Scoring:** 468+ facial landmarks analyzed for symmetry, jawline definition, orbital harmony, and vertical thirds.
- **Persistent Scan History:** Track your progress over time with custom SVG-based trend graphs.
- **30-Day AI Improvement Plans:** Metric-specific improvement strategies generated by Gemini AI.

### 🏆 Competitive Ecosystem
- **ELO Rating System:** Gain +25 for wins, climb from "Normie" to "Elite Tier."
- **Social Flex Cards:** Generate high-fidelity share cards of your wins and scans.
- **Persistent Identity:** Firebase-backed accounts to sync your rankings across devices.

### 🔊 Sensory Feedback
- **Audio:** Synthesized countdown beeps, "GO!" bursts, and triumphant fanfares.
- **Haptics:** Precise mobile vibrations for key transitions.

## 🛠 Tech Stack

- **Frontend:** [Next.js](https://nextjs.org/) (App Router), [Tailwind CSS](https://tailwindcss.com/), [Framer Motion](https://www.framer.com/motion/)
- **AI/Vision:** [MediaPipe Face Landmarker](https://developers.google.com/mediapipe/solutions/vision/face_landmarker)
- **Backend:** [Socket.io](https://socket.io/), [Firebase Auth & Firestore](https://firebase.google.com/)
- **Intelligence:** [Google Gemini API](https://ai.google.dev/)

## ⚙️ Setup & Installation

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/hussainwaseer/looksmax-clash.git
   cd looksmax-clash
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Environment Variables:**
   Create a `.env.local` file:
   ```env
   GEMINI_API_KEY=your_gemini_key_here
   ```

4. **Initialize Firebase:**
   Update `src/lib/firebase.ts` with your config.

5. **Run Development Server:**
   ```bash
   npm run dev
   ```

6. **Hosting:**
   Optimized for deployment on [Railway](https://railway.app/).
